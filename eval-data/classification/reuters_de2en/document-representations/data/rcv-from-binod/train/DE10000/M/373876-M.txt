Frankfurt ( Reuter ) - An der DTB ist der Umsatz mit Aktienoptionen am Montag wenig verändert geblieben . 
wie die DTB mitteilte , wurden bis 16.00 Uhr 37.597 ( Freitag 37.904 ) Kontrakte umgesetzt , davon 23.097 ( 22.405 ) Calls und 14.500 ( 15.499 ) Puts . 
die Call-Put-Ratio betrug 1,59 ( 1,44 ) . 
Umsatzspitzenreiter waren erneut Siemens mit 8375 Calls und 3813 Puts . 
der Umsatz mit Optionen auf den Dax sank kräftig : es wurden 30.624 ( 68.342 ) Calls und 34.231 ( 60.359 ) Puts gehandelt . 
der Dax-Volatilitätsindex fiel auf 20,55 ( 21,27 ) . 
der Juni-Dax-Future notierte um 16 Uhr nach einem Umsatz von 15.211 ( 14.733 ) Kontrakten mit 3348,5 ( 3364 ) Punkten , der Juni-MDax-Future nach einem Umsatz von 193 ( 340 ) Kontrakten mit 3565,0 ( 3580 ) Punkten . 
im Juni-Bund-Future wurden 54.792 ( 71.236 ) Kontrakte gehandelt , im Bobl-Future 35.327 ( 47.513 ) und im Juni-Schatz-Future 13.646 ( 10.679 ) Stück . 
im Einmonats-Libor-Future gingen über alle Sichten 727 ( keine ) Kontrakte um , im Dreimonats-Libor-Future waren es 1429 ( 3300 ) . 
auf den Bund-Future wurden 348 ( 499 ) Call-und 584 ( 80 ) Put-Optionen gehandelt , auf den Bobl-Future 16 ( 590 ) Calls und 1025 ( 2660 ) Puts . 
auf den US-Dollar gingen 475 ( 355 ) Call-und 907 ( 691 ) Put-Optionen um .
