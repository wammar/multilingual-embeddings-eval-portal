Frankfurt ( Reuter ) - An der DTB ist der Umsatz mit Aktienoptionen am Freitag leicht gefallen . 
wie die DTB mitteilte , wurden bis 16.00 Uhr 27.077 ( 27.946 ) Kontrakte umgesetzt , davon 15.096 ( 18.322 ) Calls und 11.981 ( 9624 ) Puts . 
die Call-Put-Ratio betrug 1,25 ( 1,90 ) . 
Umsatzspitzenreiter waren Deutsche Bank mit 3890 Calls und 2421 Puts . 
der Umsatz mit Dax-Optionen verteilte sich auf 32.106 ( 36.596 ) Calls und 31.020 ( 30.876 ) Puts . 
der Dax-Volatilitätsindex fiel auf 19,13 ( 19,14 ) . 
der Juni-Dax-Future notierte um 16 Uhr nach einem Umsatz von 13.998 ( 14.927 ) Kontrakten mit 3359 ( 3414 ) Punkten , der Juni-MDax-Future nach einem Umsatz von 44 ( 299 ) Kontrakten mit 3570 ( 3572 ) Punkten . 
im Juni-Bund-Future wurden 77.809 ( 75.600 ) Kontrakte gehandelt , im Bobl-Future 39.125 ( 53.246 ) und im Juni-Schatz-Future 5020 ( 6787 ) Stück . 
im Einmonats-Libor-Future gingen über alle Sichten 117 ( 600 ) Kontrakte um , im Dreimonats-Libor-Future waren es 4797 ( 3307 ) . 
auf den Bund-Future wurden 571 ( 2230 ) Call-und 484 ( 872 ) Put-Optionen gehandelt , auf den Bobl-Future 232 ( 500 ) Calls und 1180 ( 2520 ) Puts . 
auf den US-Dollar gingen 899 ( 2498 ) Call-und 263 ( 636 ) Put-Optionen um .
